Different types of units:

1) px(pixels) :- mostly used in give margin,padding height and width (most used unit in css) => 1px = 1/96 inch
2) %(percentage) :- used in inherit style of parent element (2nd most used unit)
3) ch(character '0') :- The ch unit in CSS that represents the width of the "0" (zero) 
    character of the element's current font. 
4) vh(view height) :- represent height of webpage 1% part (1vh = 1% of screen height/ 100vh = 100%)
5) vw(view width) :- represent width of webpage 1% part (1vw = 1% of screen width/ 100vw = 100%)
6) em(element size) :- relative font-size of current element 
7) rem(root element size) :- relative font-size of root element
